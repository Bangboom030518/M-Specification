## Boolean

- `boolean`: `true` or `false`.

## Char

- `char`: A `utf8` encoded character.

## Unit

- `Nil`: A lack of value

## Array
`

- `type[length]`: An array with items of type `type` and a fixed length of `length`.

## Slice

- `[type]`: A slice of variable length with items of type `type`.
