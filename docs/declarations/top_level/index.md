# Top Level

## Syntax

| keyword    | name         | assignment | declaration              |
| ---------- | ------------ | ---------- | ------------------------ |
| `type`     | `identifier` | `=`        | `<type_declaration>`     |
| `function` | `identifier` | `=`        | `<function_declaration>` |
| `const`    | `identifier` | `=`        | `<expression>`           |
