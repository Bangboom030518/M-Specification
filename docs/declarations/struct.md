# Struct

## Syntax

| `struct` keyword | name | `do` keyword | expression             | [`else` branch] |
| ---------------- | ------------ | ------------ | ---------------------- | --------------- |
| `struct`         | `User` | `{`         | `"Needs re-education"` |                 |

> **Note**: Without an `else` branch present, `expression` is required to evaluate to `nil`.
